# BrandAnalysisSystem
品牌分析系统主项目

## 使用版本：
- docker: Docker version 18.09.5, build e8ff056
- docker-compose: docker-compose version 1.17.0, build ac53b73

## 启动方式：

- $ bash before-start.sh
- $ docker-compose up --build --force-recreate

## 停止系统：

- $ docker-compose stop

## 停止并删除镜像

- $ docker-compose down --rmi all
